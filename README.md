# 🚀 Synergia-Connect

Synergia-Connect is a full-stack web application designed to enable seamless collaboration, communication, and networking among users. It provides a platform where individuals can connect, share ideas, and build meaningful professional relationships.

---

## 📌 Features

- 🔐 User Authentication (JWT-based secure login/signup)
- 👤 User Profiles
- 🤝 Connection / Networking System
- 💬 Messaging / Communication
- 📊 User Dashboard
- 🌐 Responsive Design

---

## 🛠️ Tech Stack

### Frontend
- HTML, CSS, JavaScript
- React.js

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Tools & Technologies
- Git & GitHub
- REST APIs
- JWT Authentication


## ⚙️ Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/your-username/synergia-connect.git
cd synergia-connect
```

### 2. Install dependencies

Backend:
```bash
cd server
npm install
```

Frontend:
```bash
cd client
npm install
```

---

### 3. Environment Variables

Create a `.env` file inside the `server` folder:

```
PORT=5000
MONGO_URI=your_database_url
JWT_SECRET=your_secret_key
```

---

### 4. Run the project

Backend:
```bash
npm start
```

Frontend:
```bash
npm start
```
